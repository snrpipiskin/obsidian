В чём разница между стеком и кучей с точки зрения GC?

---

**GC работает только с кучей (heap).** Стек (stack) вообще не управляется сборщиком мусора.

**Стек:**
- Очищается автоматически при выходе из функции
- Не требует участия GC
- Быстрое выделение и освобождение памяти

**Куча:**
- Управляется GC
- Хранит объекты с неизвестным временем жизни
- Объекты, которые переживают return из функции (escape to heap)
- Требует периодического сканирования и очистки через GC
