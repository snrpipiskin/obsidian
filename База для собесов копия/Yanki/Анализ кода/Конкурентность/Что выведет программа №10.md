```go
package main

import "fmt"

func run() {
	ch := make(chan string)
	go func() {
		for m := range ch {
			fmt.Println("processed:", m)
		}
	}()
	ch <- "cmd.1"
	ch <- "cmd.2"
}

func main() { run() }
```

---

**Объяснение:**
`main` завершается раньше чем горутина прочитает канал.

**Вывод:**
Иногда одна строка, иногда две строки, иногда без строк.
