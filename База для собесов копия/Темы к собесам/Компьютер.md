# Работа процессора

## **Конвейерная обработка инструкций**

Современные процессоры используют **конвейер** для выполнения инструкций, разделяя процесс выполнения на несколько этапов.

### **Пять этапов конвейера**

- **Instruction Fetch (IF)** — загрузка инструкции из памяти.
    
- **Instruction Decode (ID)** — декодирование инструкции и определение операндов.
    
- **Execute (EXE)** — выполнение операции (арифметика, логика и т. д.).
    
- **Memory Access (MEM)** — доступ к памяти (чтение или запись данных).
    
- **Write Back (WB)** — запись результата обратно в регистры.
