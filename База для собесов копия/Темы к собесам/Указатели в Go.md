## Что такое указатель?

**Указатель (pointer)** – это переменная, которая **содержит адрес объекта любого типа ([[obsidian/База для собесов копия/Темы к собесам/Мапа Go|Мапа Go]] , [[obsidian/База для собесов копия/Темы к собесам/Строки]], [[obsidian/База для собесов копия/Темы к собесам/Массивы & Слайсы]] и т.д.)**. **Указатель указывает на местоположение значения в памяти**. 

### Почему используются указатели?

- **Структуры и массивы копируются** при передаче в функции и присваивании.
    
- **С указателями можно передавать данные по ссылке**, а не копировать их, что экономит память.
    
- **Указатели позволяют изменять значения переменных из других функций**.

## Использование указателей

### Операторы `&` и `*`

- `&x` – **возвращает адрес переменной** `x`.
    
- `*ptr` – **разыменовывает указатель**, получая значение, на которое он указывает.
    

Пример:

```
var x int = 10
var p *int = &x // p теперь указывает на x
fmt.Println(*p) // Выведет 10
```

### Функция `new()`

- Создает новый объект и **возвращает указатель на него**.
    

```
ptr := new(int) // ptr – указатель на новый int
*ptr = 5
fmt.Println(*ptr) // 5
```

---

## Когда использовать/не использовать указатели?

**Использовать, если:**

- **Передаем большие структуры**, чтобы избежать копирования.
    
- **Хотим изменять данные внутри функции**.
    

**Не использовать, если:**

- **Работаем с простыми типами** (int, bool) – копирование быстрее, чем работа с указателями.
    
- **Работаем с краткоживущими данными**, которые не должны храниться в куче.
