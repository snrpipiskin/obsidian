Как написать параллельные HTTP-запросы на Golang?

---

Стандартный подход заключается в использовании **горутин** и **sync.WaitGroup**:
1. Объявляем `sync.WaitGroup`.
2. В цикле проходимся по всем URL.
3. Перед запуском горутины добавляем счетчик в группу: `wg.Add(1)`.
4. Запускаем горутину для каждого запроса, внутри которой выполняется HTTP-запрос, а по завершении вызывается `defer wg.Done()`.
5. В основной горутине ожидаем завершения всех запущенных горутин с помощью `wg.Wait()`.
