Что происходит после ввода google.com в браузер и нажатия Enter? (участники и этапы)

---

**Участники:**
*   **Browser (Клиент)**: парсит URL, отправляет запросы, рисует страницу
*   **DNS-серверы**: переводят домен (google.com) в IP-адрес
*   **Target Server (Сервер Google)**: принимает HTTP-запрос, отдает результат

**Процесс:**
1.  **Парсинг URL:** Браузер определяет протокол (HTTPS) и домен.
2.  **DNS-резолвинг:** Поиск IP-адреса: *Кэш браузера -> Кэш ОС -> Роутер -> ISP DNS -> Корневой DNS...*
3.  **TCP-соединение:** Установка связи с сервером по IP (3-way handshake: SYN, SYN-ACK, ACK).
4.  **TLS-handshake:** Обмен ключами шифрования (так как HTTPS).
5.  **HTTP-запрос:** Браузер шлет `GET / HTTP/2...`
6.  **HTTP-ответ:** Сервер отвечает статусом (например, `200 OK`) и отдает HTML.
7.  **Рендеринг:** Браузер строит DOM (из HTML) и CSSOM (из CSS), качает JS/картинки и отрисовывает UI.
