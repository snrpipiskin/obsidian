```go
package main

import "fmt"

type Person struct {
	Name string
}

func changeName(person *Person) {
	person = &Person{
		Name: "Alice",
	}
}

func main() {
	person := &Person{
		Name: "Bob",
	}
	fmt.Println(person.Name)
	changeName(person)
	fmt.Println(person.Name)
}
```

---

Bob
Bob

**Объяснение:**
В Go параметры передаются по значению, включая указатели. Внутри `changeName` переприсваивается **копия** указателя на новый объект, а оригинальный указатель в `main` остается без изменений.

**Исправления:**
1. Изменить поле: `person.Name = "Alice"`
2. Вернуть новый объект: `return &Person{Name: "Alice"}`
